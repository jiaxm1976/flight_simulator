# ✈️ Flight Simulator（飞行模拟器）

## 📌 项目信息

- **项目名称**：Flight Simulator
- **开发者**：Risen
- **开发开始**：2025-12-13
- **技术栈**：Babylon.js + JavaScript (Vanilla)
- **项目类型**：教学项目 / 个人兴趣项目

---

## 🎯 项目目标

从零开始开发一个基于Web的3D飞行模拟器，实现：
- ✈️ 飞机在3D空间中飞行
- 🎮 键盘控制飞行姿态
- ⚙️ 真实的飞行物理（升力、阻力、重力）
- 🛫 完成起飞和降落
